MAN|tenimiento

Software de Escritorio

El enfoque principal de este proyecto es mejorar la administración de una tienda de mantenimiento, mediante el desarrollo de una aplicación de escritorio. Esta aplicación, diseñada para ofrece una solución integral para el equipo de la tienda. Con ella, se puede llevar un registro detallado de los valores asociados a cada servicio realizado, así como un historial completo de las actividades pasadas. Además, la aplicación facilita la comunicación y coordinación del equipo, permitiendo un mejor seguimiento de los servicios y una atención más rápida y eficiente a las necesidades de los clientes.

Tienda Mascota

Aplicaciones Web

El propósito fundamental de este proyecto es mejorar la gestión de inventario, ventas y otros aspectos cruciales dentro de una tienda de mascotas mediante el uso de una plataforma web. Esta iniciativa conllevó al desarrollo de una página web diseñada para la gestión integral de la tienda, La implementación incluyó la automatización del seguimiento del inventario con cada venta realizada, la posibilidad de agregar nuevos productos directamente a través del menú administrativo sin necesidad de acceder a otras plataformas externas. Esto permitió una gestión más eficiente del stock, agilizó el proceso de ventas y ofreció mayor comodidad tanto para los clientes como para el equipo administrativo.

RegistrApp

Aplicaciones Web y Mobile

Se llevó a cabo un estudio con el objetivo de ofrecer soluciones tecnológicas que optimicen la gestión de asistencia y registro educativo. Esto se logró mediante el desarrollo de una página web destinada a los docentes, junto con la implementación de un sistema de generación de códigos QR asociados a varios factores relevantes. Al mismo tiempo, se desarrolló una aplicación móvil dirigida a los alumnos, la cual permite escanear los códigos QR para registrar la asistencia de manera instantánea en el sistema. Este enfoque integral proporciona una herramienta eficaz para mejorar la eficiencia en el seguimiento de la asistencia y el registro académico.
